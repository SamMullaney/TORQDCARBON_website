# TORQD CARBON - Premium Steering Wheel Customization

A modern, responsive website for TORQD CARBON, specializing in premium carbon fiber steering wheel customization for luxury and performance vehicles.

## Features

- **Responsive Design**: Mobile-first approach with modern CSS Grid and Flexbox
- **Dark Theme**: Carbon fiber aesthetic with neon green accents
- **Smooth Animations**: CSS transitions and JavaScript-powered scroll animations
- **Interactive Navigation**: Mobile-friendly hamburger menu with smooth scrolling
- **Contact Form**: Functional contact form with validation
- **Brand Sections**: Dedicated pages for BMW, Audi, and Mercedes steering wheels
- **Customization Tools**: Interactive customization options
- **Gallery**: Image gallery with hover effects
- **About Section**: Company information and statistics

## Pages/Sections

1. **Home/Hero** - Landing page with call-to-action
2. **BMW** - BMW steering wheel products
3. **Audi** - Audi steering wheel products
4. **Mercedes** - Mercedes steering wheel products
5. **Customize** - Customization options and tools
6. **Gallery** - Product showcase
7. **About** - Company information and stats
8. **Contact** - Contact form and information

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Grid, Flexbox, and animations
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome** - Icons
- **Google Fonts** - Inter font family

## Setup Instructions

1. **Clone or Download** the project files
2. **Open** `index.html` in a web browser
3. **No build process required** - this is a static website

## Customization

### Colors
The website uses a dark theme with these primary colors:
- Background: `#0a0a0a` (Dark)
- Accent: `#00ff88` (Neon Green)
- Text: `#ffffff` (White)
- Secondary Text: `#cccccc` (Light Gray)

### Adding Content
- **Products**: Add new product cards in the respective brand sections
- **Images**: Replace placeholder divs with actual images
- **Contact Info**: Update contact information in the contact section
- **About Content**: Modify company information and statistics
